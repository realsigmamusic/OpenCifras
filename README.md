# [OpenCifras](https://realsigmamusic.github.io/OpenCifras/)
O **OpenCifras** é uma aplicação (PWA) criada para **músicos, professores e estudantes** que querem organizar e usar cifras de forma prática, rápida e confiável — **sem internet, sem contas e sem complicação**.

Tudo funciona direto no seu dispositivo.

**Acesse agora:** [https://realsigmamusic.github.io/OpenCifras/](https://realsigmamusic.github.io/OpenCifras/)

## O que é o OpenCifras?
Uma ferramenta **leve, minimalista e objetiva** para criar, guardar e visualizar cifras e letras de músicas.

O sistema usa o formato **ChordMark**, que garante:
* Acordes sempre alinhados
* Transposição automática de tom
* Formatação inteligente

## Funcionalidades
* CRUD (Crie, edite e apague músicas com facilidade e **do seu jeito**. Tudo salvo localmente, com total privacidade.)
* Funciona Offline (Use seu repertório **em qualquer lugar**, mesmo sem internet)
* Busca por:
	* Título
	* Artista
	* Letra
* Backup e Restauração (Exporte todo o seu acervo em um único arquivo e restaure quando quiser)
* Modo Claro e Modo Escuro
* Ajuste de tamanho da fonte
* Transposer
* Metrônomo
* Tabelas de campo harmônico (maior, menor "natural, harmônica, melódica")
* Informações dos acordes
* Modos de vizualização:
	* OpenCifras
	* Simples
	* Letra
	* Acordes
* Compartilhar musica por:
	* Link
	* Texto formatado para whatsapp
* Rolagem automática
